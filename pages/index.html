TP-7 MTP

Unofficial Linux file transfer tool for the Teenage Engineering TP-7 · GitHub

Install

# Add the APT repository
curl -fsSL https://sparkyfen.github.io/tp-7-mtp/pubkey.asc \
  | sudo tee /etc/apt/keyrings/tp7-mtp.asc > /dev/null

CODENAME=$(grep UBUNTU_CODENAME /etc/os-release | cut -d= -f2)

echo "deb [signed-by=/etc/apt/keyrings/tp7-mtp.asc] \
  https://sparkyfen.github.io/tp-7-mtp $CODENAME main" \
  | sudo tee /etc/apt/sources.list.d/tp7-mtp.list

# Install
sudo apt update && sudo apt install tp7-mtp

Ubuntu 24.04 (noble) Ubuntu 22.04 (jammy) Linux Mint 21/22

Usage

Launch tp7-tray from your app menu for the system tray helper, or use the CLI:

# Switch to MTP mode and mount
sudo tp7-mtp --mount /mnt/tp7

# Copy your recordings
cp /mnt/tp7/recordings/*.wav ~/Music/

# Disconnect when done
sudo tp7-mtp --disconnect

How it works

The TP-7 connects as a USB MIDI/Audio device by default. This tool sends a proprietary MIDI SysEx command (reverse-engineered from the official Field Kit macOS app) to switch it into MTP file transfer mode, then mounts it as a regular filesystem.
